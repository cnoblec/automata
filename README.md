# automata
PHYS 349 Cellular Automata Project

## Dependencies

- numpy
- matplotlib
- typing

## Subclassing

Tests are created in `automata/tests` to demonstrate working examples of different `CA` subclasses. Explicit documentation is also included in `code/ca.py`'s documentation
